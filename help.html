1. 작성자(연구원) 사용법

매주(또는 매월) 본인 담당 과제의 활동 내용을 입력하는 분들을 위한 안내입니다.

1접속

아래 URL을 브라우저에 입력 (즐겨찾기에 등록해두면 편함):

https://koochulji.github.io/track-roadbed-report/

처음 접속하면 자동으로 로그인되며 별도 비밀번호 없음. 회사 컴퓨터에서 안 열리면 시크릿 창(Ctrl+Shift+N)으로 시도.

2본인 선택

화면에 작성자 명단이 나옵니다. 본인 이름 클릭.

본인 이름이 안 보이면 관리자에게 작성자 등록 요청. 잘못 선택했으면 상단의 🔄 본인 변경 버튼으로 다시 선택 가능.

3본인 담당 과제 자동으로 표시

본인 선택 시 시스템이 책임자 이름이 들어간 과제들을 자동으로 펼쳐줍니다. 예:

지구철 선임 → 자동으로 다음 과제들 등장:
• [기본사업] 장대레일 성능기반 유지관리 체계 구축
• [기본사업] 최적 설계기법을 활용한 궤도구성품...
• [기본사업] Agentic AI 기반 궤도 설계 자동화 기술 개발
본인이 작성할 과제가 더 있으면 화면 하단 "+ 기존 과제 선택" 드롭다운으로 추가, 또는 "🆕 새 과제 만들기" 로 즉석 등록.

4활동 내용 입력

각 과제에 두 칸이 있음:

  • [지난 주 실적] — 이번 회의에서 보고할 지난 주(달) 한 일
  • [이번 주 계획] — 다음 주(달)에 할 예정 일

각 항목마다 4가지 칸:

  • 수행내용 (필수): "레일 RCI 기법 개발 관련 회의" 같이
  • 날짜 (선택): "4/13" 또는 "~5/1"
  • 협업 기업/기관 (선택): "동아대학교"
  • 수행자명 (필수): 본인 이름
  • 중요 체크박스: 체크 시 주요 보고사항에 강조 표시 + 굵은 글씨

한 항목 다 채우면 "+ 지난 주 실적 항목 추가" 또는 "+ 이번 주 계획 항목 추가" 버튼으로 다음 항목.

5저장

입력하는 동안 자동으로 저장됩니다. 따로 저장 버튼 누를 필요 없음. 우측 미리보기에서 실시간으로 출력 형태 확인 가능.

컴퓨터를 중간에 끄거나 다른 사람과 공유하면 입력 내용이 그대로 남아있으니 주의.

6최종 제출

다 작성하셨으면 화면 하단 "최종 제출" 버튼 클릭. 제출 후엔 잠금 상태가 되며, 수정하려면 "잠금 해제" 버튼 또는 관리자에게 요청.

2. 관리자 사용법

회차를 만들고, 작성자/카테고리 관리, HWPX 파일 다운로드 등을 담당하는 관리자용 안내입니다.

1관리자 접속

https://koochulji.github.io/track-roadbed-report/admin.html

"Google로 로그인" 버튼 → 등록된 본인 Google 계정으로 인증.

관리자로 등록되지 않은 계정은 거부됩니다. UID 등록은 시스템 관리자(지구철 선임)에게 요청.

2새 회차 만들기

매주 새 보고를 받으려면:

  1. 좌측 상단 "새 회차" 또는 "다른 회차로 교체" 클릭
  2. 양식 선택: 주간 또는 월간
  3. 회의일 선택 (예: 2026-04-27 — 회의가 열리는 날짜)
  4. 화면에 자동으로 표시된 보고기간(지난주/이번주) 확인
  5. 확정 버튼 클릭
회차 확정 시 이전 회차는 자동으로 아카이브로 이동.

3작성자 명단 관리

"작성자 명단" 탭에서:

  • "디폴트 10명으로 리셋 / 시드": 궤도노반연구실 기본 10명 한 번에 등록
  • 이름 입력 후 "추가": 개별 추가
  • "삭제": 한 명씩 제거

4과제(카테고리) 관리

"카테고리 관리" 탭에서:

  • 각 과제 행에 "✏️ 수정": 종류/제목/담당자 변경
  • "삭제": 과제 제거
  • ↑ ↓ 화살표: 표시 순서 조정
  • 화면 하단 입력란으로 새 과제 추가
담당자 칸에 작성자 이름을 정확히 입력하면, 그 작성자가 본인 페이지 들어왔을 때 해당 과제가 자동으로 떠 있음. 공동 책임자는 콤마로 구분: 배영훈 책임, 지구철 선임

5제출 현황 확인 + HWPX 다운로드

"현재 회차" 탭에서:

  • 각 작성자별 제출 상태 확인 (미작성 / 임시저장 / 최종제출)
  • 특정 작성자 잠금 해제 가능
  • "HWPX 출력" 클릭 → 한글 파일 자동 생성 → 다운로드

다운로드된 파일은 한글에서 열어 본부장 활동사항 / 실장 활동사항 부분만 직접 채우면 회의자료 완성.

6아카이브 관리

"아카이브" 탭에서 지난 회차 목록 확인. 클릭하면 그 회차의 보고서 미리보기 + HWPX 다시 다운로드 가능. 잘못된 회차는 삭제 또는 활성 회차로 복원.

4. 자주 묻는 질문 (FAQ)

Q1. 페이지가 안 열려요.

회사 방화벽이 일부 도메인을 막을 수 있습니다. Ctrl+Shift+N 으로 시크릿 창에서 시도하거나 다른 브라우저(Chrome/Edge)로.

Q2. 본인 이름이 명단에 없어요.

관리자에게 "작성자 등록" 요청. 관리자는 [작성자 명단] 탭에서 추가.

Q3. 본인 담당 과제가 자동으로 안 떠요.

관리자가 그 과제의 담당자 필드에 본인 이름을 정확히 등록했는지 확인. 공동 담당자는 콤마(,)로 구분 (예: 지구철 선임, 정호연 선임). 등록 후 작성자 페이지 새로고침 (Ctrl+Shift+R).

Q4. 입력 도중 화면이 멈추거나 커서가 사라져요.

인터넷 일시 끊김 가능성. 잠시 후 자동 복구. 안 되면 새로고침 (입력 내용은 자동저장되어 있음).

Q5. 잘못 제출했어요. 다시 수정하고 싶어요.

본인 페이지에서 "잠금 해제" 버튼 클릭 → 다시 편집 가능. 안 보이면 관리자에게 요청.

Q6. 한글 파일이 깨져 보여요.

한글 2014 이후 버전 사용 권장. 휴먼명조 폰트가 PC에 없으면 자동으로 다른 폰트로 대체됨 (한컴 기본 제공).

Q7. 자동으로 뜬 카테고리를 다 안 쓰고 싶은데 어떻게 빼나요?

각 과제 우측의 "카테고리 삭제" 버튼 클릭. 단, 페이지 새로고침 시 다시 자동으로 추가됨. 영구 삭제는 관리자에게 담당자 필드 수정 요청.

Q8. 다른 사람의 입력을 볼 수 있나요?

본인 페이지 우측 "미리보기"에 모든 작성자의 합본 보고서가 실시간으로 표시됩니다. 다른 사람이 무엇을 적었는지 보면서 본인 항목 정리 가능.

문제가 계속되면 시스템 관리자(지구철 선임)에게 문의.
이 페이지는 시스템 변경 시 갱신됩니다 — 즐겨찾기 등록 권장.
